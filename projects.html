Data Analysis and Visualization with Python

Developed a Python script to fetch, process, and analyze cryptocurrency trading data from the Mandala and Binance APIs.

Key Features:

  • Data Fetching: Retrieves market data and trading pairs from Mandala’s API and average price data from Binance’s API.
  • Processing: Filters and processes trading pairs and their volumes, excludes specific assets, and computes total volume in USDT.
  • Analysis: Calculates the USD value for each trading pair and summarizes the total trading volume and number of active pairs.
  • Presentation: Utilizes Matplotlib for potential data visualization and PrettyTable for displaying results in a formatted table.

Tools and Libraries Used:

  • Python
  • requests for API interactions
  • defaultdict for data aggregation
  • matplotlib.pyplot for visualization (potentially)
  • PrettyTable for tabular display

OCR Text Capture

This project involves capturing text from a designated screen area using OCR (Optical Character Recognition). It displays the extracted text in a semi-transparent, resizable window.

Key Features:

  • Screen Capture: Captures a specified area of the screen using the Pillow library.
  • OCR Processing: Extracts text from the captured image using Tesseract OCR.
  • Text Display: Shows the processed text and cleaned items in a transparent, resizable window.
  • Hotkeys: Supports hotkeys for capturing text and toggling the display window.

Tools and Libraries Used:

  • Python
  • OCR with Tesseract
  • GUI with Tkinter
  • Screen capture and image processing with PIL and OpenCV
